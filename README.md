# CXXAudioContentAnalysis: libACA - repackaged for the Swift Package Manager

[Alexander Lerch's](https://github.com/alexanderlerch) C++ code accompanying the book ["An Introduction to Audio Content Analysis"](https://www.AudioContentAnalysis.org). The source code shows example implementations of basic approaches, features, and algorithms for music audio content analysis.

Original implementations are available at:
* [C++: ACA](https://github.com/alexanderlerch/libACA)
* [Matlab: ACA-Code](https://github.com/alexanderlerch/ACA-Code)
* [Python: pyACA](https://github.com/alexanderlerch/pyACA)
